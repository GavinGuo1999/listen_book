## 实机 / E2E 说明

- 本轮 Codex 用一次性 PowerShell 命令检查 `127.0.0.1:8000/api/health`。
- 如果后端未运行，则临时隐藏启动后端，等待健康检查通过后执行 E2E。
- E2E 结束后，如果后端是本轮临时启动的，则已停止该进程。
- Playwright 配置仍只会自动启动前端；日常手动运行 E2E 前仍需保证后端已启动。
