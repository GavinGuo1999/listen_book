# v0.5.0 人工验收清单

## 准备

1. 运行 `scripts\stop-dev.bat`。
2. 运行 `scripts\start-dev.bat`，确认 Worker、Backend、Frontend 三个窗口保持运行。
3. 打开 `http://127.0.0.1:5173/`，使用管理员账号登录并进入“管理员后台”。

## 系统状态

- [ ] 默认打开“系统状态”标签。
- [ ] 数据库连接显示正常。
- [ ] 数据库迁移显示 `20260715_0005`。
- [ ] 存储目录显示可写。
- [ ] 后台 Worker 显示在线，最近心跳会自动更新。
- [ ] 如果会话密钥或管理员密码仍是示例值，页面显示黄色提醒而不是泄露实际内容。
- [ ] 停止 Worker 后等待约 30 秒，刷新页面应显示 Worker 离线；重新启动后恢复在线。

## 书籍审核

- [ ] 搜索框可按书名或上传者过滤。
- [ ] 状态标签可切换待审批、解析失败、已拒绝和全部。
- [ ] 选择两本待审批书籍后出现批量操作栏。
- [ ] 批量批准后两本书从待审批列表移除。
- [ ] 在“全部”中展开两本书，均能看到独立审批历史。
- [ ] 批量拒绝时填写的备注能分别显示在对应书籍上。

## 用户管理

- [ ] 可按用户名搜索，并按账号状态和角色筛选。
- [ ] 点击书本图标可展开该用户的上传记录。
- [ ] 普通用户可被授予或撤销管理员权限。
- [ ] 普通用户被停用后无法登录，重新启用后可再次登录。
- [ ] 当前登录账号的权限和停用按钮不可用。
- [ ] 尝试修改 bootstrap 管理员时，页面显示保护提示。

## 任务与审计

- [ ] “任务队列”仍能按状态筛选并重试失败任务。
- [ ] “操作审计”能看到启用、停用、授予管理员和撤销管理员记录。
- [ ] 审计记录显示操作管理员、目标用户和操作时间。

## 窄屏检查

- [ ] 浏览器宽度缩小到约 390px 后，五个标签可横向滚动。
- [ ] 系统检查、批量操作栏和用户筛选改为单列且无文字遮挡。
- [ ] 用户操作图标仍可点击，并有悬停提示或无障碍名称。
