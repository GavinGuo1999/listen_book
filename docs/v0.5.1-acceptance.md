# v0.5.1 TTS 人工验收清单

## 自动检查

在项目根目录执行：

```powershell
.venv\Scripts\python.exe -m pytest backend\tests\test_text_splitter.py backend\tests\test_tts.py
.venv\Scripts\python.exe scripts\generate_tts_golden.py --dry-run
```

预期：切句黄金用例、TTS 音色/韵律用例全部通过，dry-run 输出 11 个固定样例。

## 生成试听

```powershell
.venv\Scripts\python.exe scripts\generate_tts_golden.py --force
```

打开：

```text
storage/audio/tts_golden/v0.5.1/index.html
```

逐条确认：

- 中文普通对白与旁白有轻微区分，不过度升调。
- 中文低声对白明显更慢、更低，喊话更快、更高但不刺耳。
- 中文长旁白不赶读，省略号结尾有自然收束。
- 英文样例使用英文音色，没有中文口音式拼读。
- 英文低声/喊话有可辨别但克制的差异。
- 中英混排句的中文部分保持中文音色，`The End` 使用英文音色且不出现异常长停顿。
- “她翻到第 12 页，停在 ‘The End.’ 那一行”中的“行”读作 `háng`，页面仍显示原文。
- 桌面和窄屏下试听索引没有页面级横向溢出。

评分表和口径见 `docs/tts-evaluation.md`。任何低于 3 分的项目，应记录样例 ID、问题时间点和听感后再调整规则。
